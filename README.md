<h1 align="center"> THE solution for easy and fast tweening</h2>

#### create a tween
```csharp
RectTween tween;
tween = new RectTween(start,end,tweenSetting);
Tweens.Register(tween);
```

#### step simulation
```csharp
Tweens.Step(deltaTime);
```
